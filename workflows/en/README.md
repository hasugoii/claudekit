# ClaudeKit Workflows - English

This is the English workflow collection for ClaudeKit - Enhancement Kit for Claude Code.

## Workflow List

### Planning Phase
| Command | Description |
|---------|-------------|
| `/init` | Create new project |
| `/recap` | Project summary (context recovery) |
| `/brainstorm` | Brainstorm ideas |
| `/plan` | Design features |
| `/next` | Suggest next steps |

### Building Phase
| Command | Description |
|---------|-------------|
| `/visualize` | UI/UX design |
| `/code` | Code according to Spec |
| `/run` | Run application |

### Operating Phase
| Command | Description |
|---------|-------------|
| `/debug` | Fix system errors |
| `/test` | Test system |
| `/refactor` | Refactor code |
| `/rollback` | Rollback version |
| `/deploy` | Deploy application |

### Managing Phase
| Command | Description |
|---------|-------------|
| `/save-brain` | Save project context |
| `/config` | Configure skills/agents |
| `/audit` | Security audit |
| `/ak-update` | Update ClaudeKit |
| `/uninstall` | Uninstall |
| `/cloudflare-tunnel` | Manage tunnel |
| `/customize` | Customize AI persona |

## How to Use

1. Open Claude Code in VSCode
2. Type workflow command (e.g., `/plan`)
3. Follow AI instructions

## Change Language

```
/config language en    # English
/config language vi    # Vietnamese
/config language zh    # Chinese
/config language ja    # Japanese
```
