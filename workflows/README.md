# ClaudeKit Workflows

This directory contains workflow files for ClaudeKit in multiple languages.

## Languages

| Directory | Language |
|-----------|----------|
| `en/` | English |
| `vi/` | Tiếng Việt (Vietnamese) |
| `zh/` | 中文 (Chinese) |
| `ja/` | 日本語 (Japanese) |

## Workflows

Each language contains 20 workflow files:

### Phase 1: Planning
- `init.md` - Project initialization
- `recap.md` - Context recovery
- `brainstorm.md` - Idea exploration
- `plan.md` - Feature planning
- `next.md` - Next step suggestions

### Phase 2: Building
- `visualize.md` - UI/UX design
- `code.md` - Coding workflow
- `run.md` - Run application

### Phase 3: Operating
- `debug.md` - Debug issues
- `test.md` - Testing
- `refactor.md` - Code refactoring
- `rollback.md` - Rollback
- `deploy.md` - Deployment

### Phase 4: Managing
- `save_brain.md` - Save context
- `config.md` - Configuration
- `audit.md` - Security audit
- `ak-update.md` - Update ClaudeKit
- `customize.md` - AI customization
- `cloudflare-tunnel.md` - Tunnel management
- `uninstall.md` - Uninstall

## Changing Language

To change the workflow language:

```
/config language [code]
```

Where `[code]` is one of: `en`, `vi`, `zh`, `ja`
