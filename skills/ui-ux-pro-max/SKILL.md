# UI/UX Pro Max

> Thiết kế UI/UX chuyên nghiệp

## Design Principles

### Visual Hierarchy
- Size: Lớn = quan trọng
- Color: Contrast để highlight
- Spacing: Group related items

### User Flow
- Clear call-to-actions
- Minimal steps to goal
- Feedback for actions

### Accessibility
- Color contrast 4.5:1
- Keyboard navigation
- Screen reader support

## Modern Trends

```
✅ Glassmorphism
✅ Neumorphism (subtle)
✅ Dark mode
✅ Micro-animations
✅ Gradient accents
```

## Component Styles

### Button
```css
.btn-primary {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  border-radius: 8px;
  padding: 12px 24px;
  transition: transform 0.2s, box-shadow 0.2s;
}

.btn-primary:hover {
  transform: translateY(-2px);
  box-shadow: 0 4px 12px rgba(102, 126, 234, 0.4);
}
```
