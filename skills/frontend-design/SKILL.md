# Frontend Design

> Nguyên tắc thiết kế UI/UX cho web

## Design Principles

### 1. Visual Hierarchy
- Size, color, contrast để highlight important elements
- F-pattern / Z-pattern reading

### 2. Consistency
- Design system với tokens
- Reusable components
- Predictable patterns

### 3. Whitespace
- Đừng sợ khoảng trống
- Breathing room cho content

### 4. Accessibility
- Color contrast ratio ≥ 4.5:1
- Keyboard navigation
- Screen reader support

## Component Patterns

```
Button variants: primary, secondary, ghost
Form states: default, focus, error, disabled
Card layouts: media, content, actions
Modal: header, body, footer
```
