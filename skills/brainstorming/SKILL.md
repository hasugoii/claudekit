# Brainstorming

> Kỹ thuật động não và ideation

## Techniques

### Mind Mapping
```
        ┌─ Feature A
Feature ┼─ Feature B
        └─ Feature C
```

### SCAMPER
- Substitute
- Combine
- Adapt
- Modify
- Put to other use
- Eliminate
- Reverse

### 5 Whys
```
Problem → Why? → Why? → Why? → Why? → Root Cause
```

## Questions to Ask

```
- User là ai?
- Pain point là gì?
- Giải pháp hiện tại?
- Unique value?
- MVP features?
- Nice-to-have?
- Risks?
- Dependencies?
```

## Output Format

```markdown
## Idea: [Name]
**Problem:** [What problem it solves]
**Solution:** [How it solves]
**Pros:** [Benefits]
**Cons:** [Drawbacks]
**Effort:** [Low/Medium/High]
**Impact:** [Low/Medium/High]
```
