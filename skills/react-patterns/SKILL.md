# React Patterns

> Chuyên gia React với các patterns và best practices

## Khi nào dùng
- Tạo React components
- State management
- Hooks patterns
- Performance optimization

## Patterns chính

### Component Patterns
- Compound Components
- Render Props
- Higher-Order Components
- Custom Hooks

### State Patterns
- Lifting State Up
- Context + Reducer
- Derived State
- Optimistic Updates

### Performance Patterns
- React.memo
- useMemo/useCallback
- Code Splitting
- Virtualization

## Anti-patterns tránh
- Props drilling quá sâu
- State trong component không cần
- useEffect dependencies sai
- Không cleanup side effects
