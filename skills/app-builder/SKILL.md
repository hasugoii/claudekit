# App Builder

> Xây dựng full-stack applications từ đầu

## Tech Stack Recommendations

### Web App
```
Frontend: Next.js + TypeScript + Tailwind
Backend: Express + Prisma
Database: PostgreSQL
Auth: JWT
```

### Quick Setups

```bash
# Next.js
npx create-next-app@latest --typescript

# Vite React
npm create vite@latest -- --template react-ts

# Express API
npm init && npm i express typescript prisma
```

## MVP Checklist

```
[ ] Authentication (login/register)
[ ] Core CRUD operations
[ ] Basic UI (forms, lists, detail)
[ ] Error handling
[ ] Input validation
[ ] Responsive design
```

## Development Flow

1. Database schema first
2. API endpoints
3. Frontend UI
4. Integration
5. Testing
6. Deploy
