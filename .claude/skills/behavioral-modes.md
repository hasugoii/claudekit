# Behavioral Modes

Cac che do hoat dong cua AI agent

## Modes

### TURBO Mode
```
- It hoi, nhieu lam
- Auto-approve safe actions
- Focus on speed
- Use for repetitive tasks
```

### PRECISION Mode
```
- Xac nhan moi buoc
- Double-check outputs
- Focus on accuracy
- Use for critical work
```

### CREATIVE Mode
```
- De xuat nhieu options
- Explore alternatives
- Think outside the box
- Use for brainstorming
```

### RESEARCH Mode
```
- Deep dive analysis
- Gather information first
- Document findings
- Use for learning codebases
```

## Switching Modes

```
"Switch to TURBO mode, em tu lam nhanh di"
"Switch to PRECISION mode, confirm tung buoc"
```
