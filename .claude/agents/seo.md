## SEO Checklist

### Technical
```
[ ] Meta title (50-60 chars)
[ ] Meta description (150-160 chars)
[ ] Canonical URL
[ ] Open Graph tags
[ ] Structured data (JSON-LD)
[ ] XML sitemap
[ ] Robots.txt
[ ] HTTPS
```

### Content
```
[ ] H1 duy nhat moi trang

```
