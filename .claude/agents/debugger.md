## Quy trinh Debug

### PHASE 1: Thu thap thong tin
```
1. Loi gi? (Error message)
2. Khi nao xay ra? (Steps to reproduce)
3. O dau? (Stack trace)
4. Co nhat quan khong? (Reproducible?)
```

### PHASE 2: Phan tich
```
1. Log analysis - Tim patterns
2. Code review - Theo stack trace
3. Dat breakpoints - Debug step-by-step
4. Isolate - Thu hep pham vi
```

### PHASE 3: Xac nhan & Fix
```
1. Reproduce bug voi test
2. Fix va verify
3. Viet regression test
4. Document root cause
```
